# 🎮 OSS OpenQuest Contributing Guide

안녕하세요! 
OSS OpenQuest 프로젝트에 관심을 가져주셔서 감사합니다.  
이 문서는 프로젝트에 기여하고 싶은 분들을 위한 가이드라인입니다.

---

## 1. 프로젝트 개요

**OSS OpenQuest**는 Pygame 기반의 2D 오픈소스 게임 프로젝트입니다.  
누구나 코드 수정, 맵 제작, 버그 리포트, 기능 제안 등으로 참여할 수 있습니다.

---

## 2. 기여 방법 요약

1. 리포지토리를 **Fork** 합니다.  
2. 로컬 환경에 **Clone** 합니다.  
3. 새로운 브랜치를 생성합니다.  
   ```bash
   git checkout -b feature/새기능이름

## 3. Issue 등록 방법

- 버그나 기능 제안이 있을 경우 **Issues 탭**에서 새 이슈를 생성해주세요.
- 이슈 제목은 간결하게, 내용은 자세히 작성합니다.

## 4. 브랜치 전략
- 모든 작업은 `main`이 아닌 **별도의 브랜치**에서 진행합니다.
- main은 항상 안정된 상태를 유지합니다.
- 개발을 진행하는 브랜치는 **feature/게임이름**과 같은 형태로 생성합니다.
- 작업이 완료되면 **Pull Request (PR)**를 생성하세요.

---

## 🧩 5. Pull Request 규칙
- PR 제목은 다음 형식을 권장합니다:
[Add] 새로운 기능 추가
[Fix] 버그 수정
[Docs] 문서 보완

---

## 💬 6. 문의
궁금한 점은 Issue에 남기거나, Maintainer에게 직접 문의해주세요.
