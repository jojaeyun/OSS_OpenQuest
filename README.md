# game-maze(미로 게임)

플레이어를 조종하여 적의 추적을 뿌리치고 골 지점까지 도달해야 하는 게임입니다.

## ✨ 기능
프로젝트가 제공하는 주요 기능을 나열합니다.

- 무작위 미로 생성 (DFS 기반)
- 플레이어 이동 및 충돌 감지
- 플레이어를 쫓아오는 적 (BFS 기반)
- 난이도 선택 시스템
- 적을 일시적으로 무력화 할 수 있는 아이템
- 맵에 남아있는 아이템의 개수를 알려주는 카운터

---

## 게임 방법

이동: 방향키

아이템: 적이 일시적으로 움직일 수 없고 플레이어가 적을 통과할 수 있는 상태가 된다.
