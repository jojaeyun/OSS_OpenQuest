# 전체 누적 랭킹 시스템 기능 제안

## Summary
현재 각 게임에 개별 랭킹 기능이 제안되어 있습니다.
본 제안서는  **모든 게임의 점수를 표준화하여 통합한 ‘전체 누적 랭킹 시스템’을 메인 화면에 도입하는 기능**을 제시합니다.

## Motivation
프로젝트 내에 다양한 게임이 존재하지만,  
사용자는 **각 게임별 랭킹만 확인할 수 있어 전체 플레이 성과를 한눈에 보기 어렵습니다.**

게임 점수를 단일 기준으로 비교할 수 있다면,
- 전체 순위를 경쟁적으로 확인할 수 있고
- 여러 게임을 플레이하는 동기가 생기며
- 프로젝트 전체적인 경험이 강화됩니다.

## Proposal
### 1. 게임 점수 표준화
각 게임마다 난이도와 점수 체계가 다르므로  
`0 ~ 100` 범위로 표준화된 점수로 변환합니다(단순 예시입니다.).

예시: NormalizedScore = (UserScore / MaxPossibleScore) * 100

### 2. 누적 점수 시스템 도입
- 각 게임의 점수를 표준화한 후 합산하여 전체 점수 생성  
- 예: TotalScore = Breakout(85) + RSP(70) + Maze(90) + ...

### 3. 메인 화면에 전체 랭킹 메뉴 추가
- "전체 랭킹" 버튼 추가  
- 상위 N명 리스트 표시  
- 각 사용자별 게임별 점수 breakdown까지 제공하면 좋을 거 같습니다.
